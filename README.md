## Hardware-Agnostic HVAC Surrogate Model (Live Inference)
This micro-project demonstrates the deployment of a Fourier Neural Operator inference pipeline tailored for thermal boundary conditions and Model Predictive Control (MPC) applications.

The core math relies on the modular architecture built at [SciML-Thermal-Core](https://github.com/Saeid-Bahrami/SciML-Thermal-Core).
